# Centered Div

This is a simple centered div React component.

## Installation

```bash
npm install centered-div
```

## Usage

```javascript
import CenteredDiv from 'centered-div';

const App = () => (
  <CenteredDiv>
    <h1>Hello World!</h1>
  </CenteredDiv>
);

export default App;
```

## Options

No options required.
